# Система подписок с биллингом

## Контекст

Стриминговая платформа с 5M+ активных пользователей. Бизнес хочет запустить систему платных подписок с несколькими тарифами, пробными периодами и интеграцией с платёжными провайдерами.

## Бизнес-цели
1. Увеличить выручку на 30% за счёт платных подписок
2. Снизить отток пользователей через гибкую систему тарифов
3. Автоматизировать биллинг и продление подписок
4. Интегрироваться с 2+ платёжными провайдерами (Stripe, ЮKassa)

## Ключевые метрики успеха
- Conversion rate (free -> paid): >5%
- Churn rate: <10% в месяц
- Payment success rate: >95%

## Роль аналитика
- Сбор требований от Product Owner и стейкхолдеров
- Проектирование архитектуры решения
- API-дизайн и спецификации
- Моделирование БД
- Написание ТЗ для разработки
- Контроль реализации
